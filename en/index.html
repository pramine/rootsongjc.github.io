Cloud Native is eating the world

New technologies such as containers, Kubernetes, Service Mesh, and serverless have emerged endlessly. Cloud computing has developed into an application-centric cloud-native stage. The cloud on the enterprise is unstoppable. What should ordinary developers do?

The momentum of the cloud is unstoppable. Developers should conform to and embrace cloud native, make the cloud work for ourslef, maximize the advantages of the cloud, and well realize the business value for developers. Let’s start with Kubernetes.
